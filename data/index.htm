Instructions

Press the button to set the duration and degrees of movement for the servo
The servo will go back and forth between the 90° point and return to 0° after the set duration of movement

Press the Add button to add a new trigger moment to your list.
You can set up to 8 seperate trigger moments.

To edit a set moment simply touch the time u wish to edit.
If u wish to remove it comletely, set the moment to an empty value
